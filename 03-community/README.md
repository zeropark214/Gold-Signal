# 금 투자자 커뮤니티

## 목적

금 투자자들이 게시글과 댓글을 통해 의견을 나눌 수 있는 커뮤니티 기능을 제공한다.

토스증권 커뮤니티의 공개 사례를 참고해, 투자자들이 매일 방문하고 반복적으로 의견을 교환할 수 있는 구조를 만든다. 단, 토스증권의 UI, 문구, 브랜드 자산, 비공개 정책을 그대로 복제하지 않고 금 투자 서비스에 맞게 재해석한다.

## 주요 범위

- 게시글 목록
- 게시글 작성
- 게시글 상세
- 댓글 작성 및 조회
- 공감
- 신고

## MVP 상세 기획 제안

커뮤니티 탭은 사용자가 금 시장에 대한 다른 투자자의 반응을 빠르게 훑고, 필요할 때 바로 의견을 남길 수 있는 화면으로 설계한다. MVP에서는 기능을 과하게 넓히기보다 `작성`, `정렬`, `작성자 식별`, `반응`, `댓글`의 기본 루프를 안정적으로 만든다.

### 참여 권한 원칙

- 게시글 목록과 상세 조회는 비로그인 사용자도 볼 수 있다.
- `새 글 작성`, 댓글 작성, 공감, 신고는 로그인한 사용자만 사용할 수 있다.
- 커뮤니티에 처음 참여하는 사용자는 회원가입 또는 로그인 후 닉네임 설정을 완료해야 한다.
- 닉네임이 없는 사용자가 참여 액션을 시도하면 닉네임 설정 화면으로 이동시킨다.
- 닉네임은 게시글과 댓글의 작성자명으로 노출되므로, 가입 단계에서 중복 확인과 금칙어 검사를 수행한다.

### 1. 새 글 작성

- 커뮤니티 상단에 `새 글 작성` 버튼을 고정 진입점으로 제공한다.
- 작성 폼에는 제목, 본문, 토론방 선택을 기본 필드로 둔다.
- 토론방 기본값은 `전체`가 아니라 사용자가 최근 보고 있던 토론방 또는 `실시간 시황`으로 둔다.
- 제목과 본문은 필수이며, 공백만 입력한 경우 등록할 수 없다.
- 등록 후에는 목록 최상단에 즉시 반영하고, 작성 완료 토스트를 표시한다.
- 로그인하지 않은 사용자가 작성 버튼을 누르면 회원가입 또는 로그인 화면으로 이동시킨다.
- 로그인했지만 닉네임이 없는 사용자는 닉네임 설정 화면으로 이동시킨다.

### 2. 최신순/인기순 필터링

- 목록 상단에는 현재 정렬값을 보여주는 단일 정렬 버튼을 제공하고, 버튼을 누를 때마다 `최신순`과 `인기순`을 바로 전환한다.
- 기본 정렬은 `최신순`으로 한다. 커뮤니티 첫 진입 시 사용자가 지금 올라오는 금 시장 반응을 확인하는 것이 가장 중요하기 때문이다.
- `최신순`은 기본 피드이자 실시간 대화 확인용 정렬이다.
- `최신순`은 작성 시각 기준 내림차순으로 정렬하고, 댓글이 달려도 게시글 자체 순서는 바꾸지 않는다.
- `인기순`은 발견용 보조 정렬이다. 사용자가 반응이 큰 글이나 중요한 토론만 빠르게 확인하고 싶을 때 사용한다.
- `인기순`은 최근 24시간 기준 공감 수, 댓글 수, 조회 수, 최근 반응 속도를 합산해 산정한다.
- 인기 점수는 `공감 수 * 3 + 댓글 수 * 5 + 최근 1시간 반응 보너스 - 시간 감쇠` 방식으로 시작한다.
- 오래된 글이 계속 상단에 머물지 않도록 인기순에는 시간 감쇠를 적용한다.
- 신고 누적 게시글은 인기순 점수를 낮추거나 인기순 노출에서 제외한다.
- 정렬을 바꾸면 목록 위치는 상단으로 이동하고, 현재 선택된 정렬값을 버튼 라벨에 명확히 표시한다.
- 커뮤니티 규모가 커지면 `오늘 인기`, `주간 인기`처럼 인기순 기간 필터를 확장할 수 있다.

### 3. 작성자 닉네임 노출

- 게시글 목록에는 작성자 닉네임을 제목 아래 보조 정보로 표시한다.
- 닉네임 옆에는 선택적으로 투자자 배지를 표시한다.
- 닉네임이 삭제 계정이거나 탈퇴 회원인 경우 `알 수 없는 사용자`로 표시한다.
- 닉네임은 사용자의 신원을 보증하지 않으므로, 배지와 함께 투자 조언 신뢰도를 과장하지 않는다.

### 4. 공감수

- 게시글 목록과 상세 화면에 공감 수를 표시한다.
- 사용자는 게시글당 한 번만 공감할 수 있다.
- 공감 버튼을 다시 누르면 공감이 취소된다.
- 본인이 작성한 글에도 공감할 수 있게 둘지 여부는 정책 결정이 필요하다. MVP 제안은 본인 글 공감 허용, 단 인기 점수에는 낮은 가중치를 적용하는 방식이다.
- 공감 수는 즉시 UI에 반영하되, 서버 반영 실패 시 원래 값으로 되돌리고 오류 메시지를 표시한다.

### 5. 댓글 기능

- 게시글 상세 화면에서 댓글 목록과 댓글 입력창을 제공한다.
- 목록 화면에는 댓글 수만 표시하고, 댓글 작성은 상세 화면에서만 허용한다.
- 댓글은 기본적으로 작성 시각 오름차순으로 보여 대화 흐름을 유지한다.
- 댓글 등록 후 입력창을 비우고 목록 하단에 즉시 반영한다.
- 댓글에는 작성자 닉네임, 작성 시간, 본문, 신고 메뉴를 표시한다.
- 대댓글은 MVP 범위에서 제외하고, 추후 필요할 때 확장한다.

### 권장 MVP 우선순위

1. 게시글 목록: 최신순/인기순, 닉네임, 공감 수, 댓글 수 표시
2. 새 글 작성: 제목/본문/토론방 입력, 등록 후 목록 반영
3. 게시글 상세: 본문, 공감 버튼, 댓글 목록
4. 댓글 작성: 입력/등록/즉시 반영
5. 안전장치: 신고, 금칙어, 반복 작성 제한

## 토스증권 커뮤니티 참고 포인트

- 종목 단위로 대화가 모이는 구조
- 주주 인증 배지를 통한 신뢰도 표시
- 게시글과 댓글 중심의 빠른 대화
- 관심 종목 팔로우를 통한 재방문 유도
- 인기 종목과 인기 게시글 노출
- 악성 글 탐지, 신고, 운영 정책 등 안전장치
- 콘텐츠 공유와 외부 노출 범위 설정

## Gold Signal 적용 방향

토스증권의 종목 중심 커뮤니티를 금 투자 앱에 맞게 아래처럼 변환한다.

| 토스증권형 패턴 | Gold Signal 적용 |
| --- | --- |
| 종목별 커뮤니티 | 금 현물, 금 ETF, 금 선물, 금 관련주, 경제 지표별 토론방 |
| 주주 인증 배지 | 금 투자자 인증 배지, ETF 보유 인증 배지, 관심 투자자 배지 |
| 관심 종목 팔로우 | 관심 토론방 팔로우 |
| 인기 종목 | 인기 토론방, 많이 언급된 지표, 많이 언급된 뉴스 |
| 게시글/댓글 중심 피드 | 실시간 시황 피드, 뉴스 토론 피드, 지표 발표 토론 피드 |
| 안전장치 | 신고, 숨김, 금칙어, 악성 글 탐지, 투자 조언 경고 |

## 배지 체계

- 금 투자자 인증 배지: 사용자가 금 관련 자산 보유를 인증한 경우 표시
- ETF 투자자 배지: 금 ETF 관심 또는 보유 인증 사용자를 표시
- 활발한 참여자 배지: 일정 기간 게시글과 댓글 활동이 많은 사용자에게 표시
- 신규 투자자 배지: 가입 초기 또는 초보 투자자임을 표시

배지는 사용자 신뢰도를 보조하는 장치이며, 투자 실력이나 수익률을 보증하지 않는다.

## 안전장치

- 게시글 신고
- 댓글 신고
- 차단 또는 숨김
- 금칙어 필터
- 반복 도배 제한
- 허위정보 의심 게시글 검토
- 투자 권유성 문구 경고
- 운영자 삭제 및 이용 제한
- 커뮤니티 운영 정책 고지
