# 금 투자자 커뮤니티

## 목적

금 투자자들이 게시글과 댓글을 통해 의견을 나눌 수 있는 커뮤니티 기능을 제공한다.

토스증권 커뮤니티의 공개 사례를 참고해, 투자자들이 매일 방문하고 반복적으로 의견을 교환할 수 있는 구조를 만든다. 단, 토스증권의 UI, 문구, 브랜드 자산, 비공개 정책을 그대로 복제하지 않고 금 투자 서비스에 맞게 재해석한다.

## 주요 범위

- 게시글 목록
- 게시글 작성
- 게시글 상세
- 댓글 작성 및 조회
- 공감
- 신고

## 토스증권 커뮤니티 참고 포인트

- 종목 단위로 대화가 모이는 구조
- 주주 인증 배지를 통한 신뢰도 표시
- 게시글과 댓글 중심의 빠른 대화
- 관심 종목 팔로우를 통한 재방문 유도
- 인기 종목과 인기 게시글 노출
- 악성 글 탐지, 신고, 운영 정책 등 안전장치
- 콘텐츠 공유와 외부 노출 범위 설정

## Gold Signal 적용 방향

토스증권의 종목 중심 커뮤니티를 금 투자 앱에 맞게 아래처럼 변환한다.

| 토스증권형 패턴 | Gold Signal 적용 |
| --- | --- |
| 종목별 커뮤니티 | 금 현물, 금 ETF, 금 선물, 금 관련주, 경제 지표별 토론방 |
| 주주 인증 배지 | 금 투자자 인증 배지, ETF 보유 인증 배지, 관심 투자자 배지 |
| 관심 종목 팔로우 | 관심 토론방 팔로우 |
| 인기 종목 | 인기 토론방, 많이 언급된 지표, 많이 언급된 뉴스 |
| 게시글/댓글 중심 피드 | 실시간 시황 피드, 뉴스 토론 피드, 지표 발표 토론 피드 |
| 안전장치 | 신고, 숨김, 금칙어, 악성 글 탐지, 투자 조언 경고 |

## 커뮤니티 카테고리

- 전체
- 실시간 시황
- 금 현물
- 금 ETF
- 금 선물
- 금 관련주
- 경제 지표 토론
- 뉴스 토론
- 질문
- 장기투자

## 배지 체계

- 금 투자자 인증 배지: 사용자가 금 관련 자산 보유를 인증한 경우 표시
- ETF 투자자 배지: 금 ETF 관심 또는 보유 인증 사용자를 표시
- 활발한 참여자 배지: 일정 기간 게시글과 댓글 활동이 많은 사용자에게 표시
- 신규 투자자 배지: 가입 초기 또는 초보 투자자임을 표시

배지는 사용자 신뢰도를 보조하는 장치이며, 투자 실력이나 수익률을 보증하지 않는다.

## 안전장치

- 게시글 신고
- 댓글 신고
- 차단 또는 숨김
- 금칙어 필터
- 반복 도배 제한
- 허위정보 의심 게시글 검토
- 투자 권유성 문구 경고
- 운영자 삭제 및 이용 제한
- 커뮤니티 운영 정책 고지
