# 유저 플로우

## 1. 신규 사용자 가입 플로우

1. 앱 실행
2. 온보딩 화면 진입
3. 서비스 핵심 기능 확인
4. 로그인 또는 회원가입 선택
5. 이메일 또는 소셜 계정으로 가입
6. 닉네임 설정
7. 푸시 알림 권한 요청
8. 관심 알림 카테고리 선택
9. 홈 화면 진입

## 2. 기존 사용자 진입 플로우

1. 앱 실행
2. 로그인 상태 확인
3. 홈 화면 진입
4. 최신 경제 지표 요약, 주요 뉴스, 인기 커뮤니티 글 확인

## 3. 경제 지표 확인 플로우

1. 하단 탭에서 `지표` 선택
2. `매일 봐야 할 것` 또는 `매월 봐야 할 것` 선택
3. 우선순위 순서로 지표 목록 확인
4. 특정 지표 선택
5. 지표 상세 화면 진입
6. 최신 수치, 이전 수치, 예상치, 발표 일시 확인
7. 금값 영향 요약 확인
8. 관련 뉴스 또는 커뮤니티 글로 이동

## 4. 매일 봐야 할 지표 확인 플로우

1. 하단 탭에서 `지표` 선택
2. `매일 봐야 할 것` 탭 선택
3. 미국 10년물 실질금리, 미국 10년물 국채금리, 달러지수 DXY, 원/달러 환율, 유가, 금 ETF 흐름 확인
4. 전일 대비 변화 방향 확인
5. 금값 영향 요약 확인
6. 필요한 경우 지표 상세 화면으로 이동

## 5. 매월 봐야 할 지표 확인 플로우

1. 하단 탭에서 `지표` 선택
2. `매월 봐야 할 것` 탭 선택
3. PCE, CPI, 고용보고서, 소매판매, ISM, GDP, 중앙은행 금 매입 지표 확인
4. 발표 예정일 또는 최신 발표 결과 확인
5. 예상치 대비 결과 확인
6. 금값 영향 요약 확인
7. 필요한 경우 지표 상세 화면으로 이동

## 6. 뉴스 속보 확인 플로우

1. 사용자가 푸시 알림 수신
2. 푸시 알림 클릭
3. 앱 실행 또는 뉴스 상세 화면으로 이동
4. 뉴스 헤드라인과 요약 확인
5. 원문 보기 버튼 선택
6. 외부 브라우저 또는 앱 내 웹뷰에서 뉴스 원문 확인
7. 앱으로 돌아와 관련 커뮤니티 의견 확인

## 7. 뉴스 목록 탐색 플로우

1. 하단 탭에서 `뉴스` 선택
2. 최신 뉴스 헤드라인 목록 확인
3. 중요도, 카테고리, 시간 기준으로 뉴스 확인
4. 뉴스 선택
5. 뉴스 상세 화면 진입
6. 요약, 출처, 발행 시간 확인
7. 원문 보기 또는 관련 커뮤니티 글 보기 선택

## 8. 커뮤니티 게시글 작성 플로우

1. 하단 탭에서 `커뮤니티` 선택
2. 게시글 목록 확인
3. 글쓰기 버튼 선택
4. 카테고리 선택
5. 제목 입력
6. 본문 입력
7. 작성 완료 선택
8. 게시글 상세 화면으로 이동

## 9. 커뮤니티 댓글 작성 플로우

1. 커뮤니티 게시글 선택
2. 게시글 상세 화면 진입
3. 기존 댓글 확인
4. 댓글 입력
5. 등록 선택
6. 댓글 목록에 즉시 반영

## 10. 신고 플로우

1. 게시글 또는 댓글의 더보기 메뉴 선택
2. 신고 선택
3. 신고 사유 선택
4. 신고 제출
5. 신고 접수 완료 메시지 표시

## 11. 푸시 알림 설정 플로우

1. 하단 탭에서 `설정` 또는 마이페이지 선택
2. 알림 설정 진입
3. 전체 푸시 알림 ON/OFF
4. 뉴스 속보 알림 ON/OFF
5. 경제 지표 발표 알림 ON/OFF
6. 커뮤니티 댓글 알림 ON/OFF
7. 설정 저장
